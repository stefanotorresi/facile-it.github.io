PHP coding standard: PSR-7 HTTP message interfaces

23 March 2015 Michele Carino

6 minute read

Le specifiche PSR-7 descrivono una proposta di standardizzazione delle interfacce per i messaggi HTTP. Come sappiamo, il protocollo HTTP, attraverso le specifiche redatte dal W3C, definisce una serie di regole di comunicazione che vengono implementate dalle applicazioni client e server che lo adottano. A prima vista niente di nuovo sotto il Sole dunque, ma cerchiamo insieme di comprendere l’insieme di problematiche che sono al centro di questo nuovo dibattito che riguarda gli standard di codifica del linguaggio PHP.
